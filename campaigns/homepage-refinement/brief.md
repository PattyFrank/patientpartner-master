# Campaign Brief: Homepage Refinement

## Goal
Refine PatientPartner.com homepage landing page copy using direct-response principles and brand memory. Improve conversion, clarity, and alignment with positioning.

## Source
- Firecrawl scrape of patientpartner.com (homepage, how-it-works, about-us)
- Brand memory: voice-profile.md, positioning.md, audience.md
- Site map: 40+ pages mapped

## Positioning Angle
Peer-to-Peer Beats Everything. "They get patients to the door. We get them through it."

## Key Changes from Current Site
1. **Hero:** Sharper headline options; removed generic "Empower your patients"
2. **Problem:** Quantified pain ($22B, 3–8% utilization); named "missing middle"
3. **Proof:** Added 71%, 73% from research; fixed "Measureable" typo
4. **Voice:** Removed "revolutionize," "comprehensive" (avoid list)
5. **Competitive wedge:** Real-time vs. coordinator-driven; "Stories build belief. Mentorship changes behavior."

## Variants Delivered
- **Control:** Full sequence, Peer-to-Peer angle
- **Contrarian:** "Engagement theater" challenge, ROI-focused
- **Proof-Led:** Numbers first, minimal warmup

## B2B Enterprise Refresh (2026-02-26)
- Applied B2B enterprise patterns: scannable structure, committee-ready proof, internal-stakes framing
- Removed "game-changer" from Sara W. testimonial (voice avoid)
- Section headlines stand alone for skim-readers
- Urgency: organizational timing only (no fake scarcity)
- Learnings applied: avoided rubric-driven cookie-cutter fixes

## Status
Draft. Ready for review and A/B testing.
